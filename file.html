Method Purpose Typical Status Codes Description Idempotent Typical Use Case
GET Retrieve or fetch data from the server 200 OK, 204 No Content, 404 Not Found
  • 200 OK: Request succeeded, and the resource is returned.
  • 204 No Content: Request succeeded, but no content to return.
  • 404 Not Found: Resource does not exist.
 Yes Fetching resource details, querying data
POST Submit data to create or process a resource 201 Created, 200 OK, 400 Bad Request, 409 Conflict
  • 201 Created: A new resource was created.
  • 200 OK: Request succeeded, and some response is returned.
  • 400 Bad Request: Request is malformed or invalid.
  • 409 Conflict: Conflict with the current state of the resource.
 No Creating new resources, submitting forms
PUT Update or replace an existing resource 200 OK, 204 No Content, 400 Bad Request, 404 Not Found
  • 200 OK: Resource was updated successfully.
  • 204 No Content: Request succeeded, but no content to return.
  • 400 Bad Request: Request is malformed.
  • 404 Not Found: Resource to update does not exist.
 Yes Replacing or updating a resource entirely
DELETE Remove a resource from the server 200 OK, 204 No Content, 404 Not Found, 409 Conflict
  • 200 OK: Resource was deleted.
  • 204 No Content: Request succeeded, but no content to return.
  • 404 Not Found: Resource to delete does not exist.
  • 409 Conflict: Conflict with the current state of the resource.
 Yes Deleting a resource
PATCH Apply partial updates to an existing resource 200 OK, 204 No Content, 400 Bad Request, 404 Not Found
  • 200 OK: Resource was partially updated.
  • 204 No Content: Request succeeded, but no content to return.
  • 400 Bad Request: Request is malformed.
  • 404 Not Found: Resource to update does not exist.
 Not necessarily Partial updates to a resource
